delonix secret

Cofre de segredos cifrado em repouso — o produtor do `run --secret`.

Um cofre local (SecretStore) cifrado com XChaCha20-Poly1305. Os valores NUNCA são impressos por omissão (redigidos; --reveal é opt-in). É a fonte dos container run --secret/--secret-files e do --password-secret do storage — o segredo entra uma vez, nunca fica no histórico do shell nem no manifesto.

Cofre de segredos (cifrado em repouso) — o produtor do `run --secret`

Usage: delonix secret [OPTIONS] <COMMAND>

Commands:
  create      Cria/substitui um segredo a partir de literais e/ou um ficheiro `.env`
  ls          Lista os segredos (nome + nº de chaves; valores NUNCA mostrados)
  inspect     Mostra as chaves de um segredo (valores redigidos, salvo `--reveal`)
  set         Define/actualiza chaves num segredo (cria-o se não existir)
  unset       Remove uma chave de um segredo (ou o segredo todo com `--all`)
  rm          Remove um segredo
  rotate-key  Roda a chave-mestra do host: re-cifra TODOS os segredos com uma chave nova. Os valores são preservados
  help        Print this message or the help of the given subcommand(s)

Options:
      --l18n <en|pt>  Output language: `en` (default) or `pt` (Portuguese, pt_AO). Also settable via `$DELONIX_L18N`. Global — works before any subcommand
  -h, --help          Print help

secret create

Cria/substitui um segredo a partir de literais e/ou um ficheiro `.env`

Usage: delonix secret create [OPTIONS] <NAME>

Arguments:
  <NAME>  

Options:
      --from-literal <FROM_LITERAL>    Par `KEY=value`. Repetível
      --l18n <en|pt>                   Output language: `en` (default) or `pt` (Portuguese, pt_AO). Also settable via `$DELONIX_L18N`. Global — works before any subcommand
      --from-env-file <FROM_ENV_FILE>  Carrega linhas `KEY=value` de um ficheiro (ex.: `.env`)
  -h, --help                           Print help

Exemplos

Criar um segredo (valor via stdin, não no argv)
printf 's3nha' | delonix secret create db-pass
